1题

1. 答案："undefined"。 然 foo.bar 传递给了函数，但是真正执行的时候，函数 bar 的上下文环境是 arguments。所以上下文环境是 arguemnts 就没问题了。

2题

答案："undefined"。new写了两次。第1个new上 function 执行后返回一样的function。不过下个new不知道用哪个function。因为有二义性，所以答案是"undefined"。

3题

答案："function"。原式变成下式。

function bar(){

  function foo(){}

    return foo;

  foo=10;

  var foo='11'}

call bar时，'bar'还没执行。所以输出是"function"。

4题

答案：3,1。 原式变成下列式。

baz: {

  x: 1,

    bar: function() {

  return this.x;

   }

}

baz.bar()执行之前，用"this"的参数来表示"baz"。所以"this.x"是1。不过参照bar函数时，"captured()"已执行，所以变量3输出。 所以答案是3,1。

5题

5.答案： "undefined" 文法不对。所以return需改成return;。所以答案是undefined。
